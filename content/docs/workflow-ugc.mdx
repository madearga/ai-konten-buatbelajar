# Workflow UGC Ads — Step by Step

## Overview

Tutorial ini memandu kamu membuat **UGC ads dari nol** menggunakan AI. Dua workflow tersedia: **Quick** (CreateUGC) dan **Pro** (Nano Banana + ComfyUI).

<Tabs items={['Quick Workflow', 'Pro Workflow']}>
  <Tab>
    ## Quick: CreateUGC (30 menit)
    
    <Steps>
    
    ### Siapkan Product Info
    Kumpulkan: product name, key benefits, target audience, dan product images (atau URL produk).
    
    ### Pilih Platform
    Login ke **CreateUGC** atau **Topview**. Untuk market Indonesia, Topview lebih cocok karena support Bahasa Indonesia.
    
    ### Input Produk
    - **CreateUGC:** Paste URL produk → AI auto-extract
    - **Topview:** Upload gambar produk + deskripsi manual
    
    ### Pilih Avatar
    Filter berdasarkan:
    - Gender dan usia yang cocok dengan target audience
    - Etnis (untuk market Indonesia, pilih Southeast Asian)
    - Style (casual, professional, sporty)
    
    ### Tulis Script
    Struktur UGC ad yang efektif:
    
    ```
    [HOOK - 3 detik]: "Eh, kamu punya masalah [X] juga?"
    [PROBLEM - 5 detik]: "Dulu aku juga kayak gitu..."
    [SOLUTION - 8 detik]: "Tapi sejak pakai [produk]..."
    [CTA - 3 detik]: "Cek link di bio sekarang!"
    ```
    
    Generate **3-5 variasi hook** untuk testing.
    
    ### Generate
    Klik generate. Tunggu ~5-10 menit. Download hasilnya.
    
    ### Quality Check
    Periksa setiap video:
    - ✅ Lip sync akurat?
    - ✅ Produk terlihat natural?
    - ✅ Script sesuai?
    - ✅ Lighting konsisten?
    
    ### Deploy
    Upload ke ad platform (TikTok Ads, Meta Ads). Test **3-5 variasi** dengan budget kecil ($5-10/hari per variasi).
    
    </Steps>
  </Tab>
  <Tab>
    ## Pro: Nano Banana 2 + ComfyUI (2-3 jam)
    
    <Steps>
    
    ### Step 1: Character Creation (Nano Banana 2)
    
    Buat karakter AI menggunakan prompt JSON terstruktur. Gunakan template dari **@IamEmily2050** sebagai starting point.
    
    ```json
    {
      "character_name": "Rina",
      "character_profile": {
        "age": 25,
        "build": "slim, average height",
        "ethnicity": "Indonesian",
        "hair": "black, long, straight",
        "features": "warm smile, brown eyes"
      },
      "wardrobe": {
        "top": "white casual t-shirt",
        "style": "minimalist, everyday wear"
      },
      "pose": "looking at camera, friendly expression",
      "environment": {
        "location": "modern Indonesian living room",
        "lighting": "natural window light, soft shadows"
      },
      "camera": {
        "type": "smartphone selfie",
        "angle": "eye level",
        "distance": "medium close-up"
      }
    }
    ```
    
    Generate **4-8 shot** dengan pose dan angle berbeda. Pilih yang terbaik.
    
    **Referensi:** Yaroflasher — "Hyper-Realistic AI Influencers in 5 Minutes" (3.4K views, 18 chapters)
    
    ### Step 2: Product Integration
    
    Untuk UGC, karakter perlu **"memegang"** produk. Ada beberapa cara:
    
    | Metode | Tools | Kualitas | Kesulitan |
    |--------|-------|----------|-----------|
    | **Inpainting** | ComfyUI + Flux | ⭐⭐⭐⭐ | ⭐⭐⭐ |
    | **IP-Adapter** | ComfyUI | ⭐⭐⭐⭐⭐ | ⭐⭐⭐⭐ |
    | **Virtual Try-On** | Higgsfield / IDM-VTON | ⭐⭐⭐⭐ | ⭐⭐ |
    | **Post-composite** | Photoshop / Canva | ⭐⭐⭐ | ⭐ |
    
    **Untuk pemula:** Mulai dengan Higgsfield Virtual Try-On atau post-composite di Canva.
    
    **Referensi:** Higgsfield — "I Created a $1,000,000 Brand Using AI" (Virtual Try-On chapter)
    
    ### Step 3: Image-to-Video
    
    Konversi karakter images menjadi video menggunakan:
    
    - **Seedance 2.0** — terbaik untuk realistic motion + lip sync
    - **Kling 3.0** — tercepat, cukup untuk social media
    - **ComfyUI + WAN 2.2** — gratis, full control
    
    Upload image karakter → tulis motion prompt → generate 5-10 detik video.
    
    **Motion prompt contoh:**
    ```
    Woman holds product bottle, smiles, nods slightly,
    turns to show product to camera, natural hand movement
    ```
    
    ### Step 4: Lip Sync (InfiniteTalk / ComfyUI)
    
    Untuk karakter yang "berbicara":
    
    1. Record voice over (bisa pakai AI TTS seperti ElevenLabs)
    2. Load video + audio ke InfiniteTalk workflow di ComfyUI
    3. AI generate lip sync secara otomatis
    
    **Referensi:** Yaroflasher — Lesson: "InfiniteTalk Video Lip Sync Workflow"
    
    ### Step 5: Audio & Music
    
    Tambahkan:
    - **Voice over** — ElevenLabs / Bark / native TTS
    - **Background music** — Suno AI / Udio / royalty-free
    - **Sound effects** — gunakan library gratis (freesound.org)
    
    ### Step 6: Edit & Finalisasi
    
    Gabungkan semua elements di editor:
    - **CapCut** (free, mobile-friendly) — untuk quick edits
    - **DaVinci Resolve** (free, professional) — untuk kontrol penuh
    - **Premiere Pro** — standar industri
    
    Perhatikan:
    - Format 9:16 untuk TikTok/Reels
    - Hook harus kuat di **3 detik pertama**
    - Subtitle/captions wajib (80% viewer tanpa suara)
    
    ### Step 7: Export & Deploy
    
    Export dalam format:
    - **TikTok/Reels:** 9:16, 1080x1920, H.264
    - **Feed Ads:** 1:1 atau 4:5
    - **YouTube:** 16:9, 1920x1080
    
    </Steps>
  </Tab>

## Checklist UGC Ads

<Callout type="warning" title="QA Checklist — Sebelum Deploy">
- [ ] Hook menarik di 3 detik pertama
- [ ] Produk terlihat jelas dan natural
- [ ] Script sesuai brand tone & messaging
- [ ] Lip sync akurat (tidak "off")
- [ ] Lighting konsisten antar shot
- [ ] Subtitle/captions ditambahkan
- [ ] CTA jelas dan actionable
- [ ] Format sesuai platform target
- [ ] Durasi sesuai best practice (15-30 detik untuk TikTok)
- [ ] Sudah test minimal 3 variasi hook
</Callout>

## Prompt Cheat Sheet

### Untuk Character (Nano Banana 2)

```
{
  "scene": "[setting — coffee shop, bedroom, gym, dll]",
  "subject": {
    "type": "[age, gender, ethnicity]",
    "hair": "[warna, panjang, style]",
    "expression": "[smile, serious, surprised, dll]",
    "pose": "[holding phone, looking at camera, dll]"
  },
  "wardrobe": {
    "top": "[deskripsi outfit]",
    "accessories": "[kalung, kacamata, dll]"
  },
  "camera": {
    "type": "[smartphone, DSLR, dll]",
    "angle": "[eye level, slightly above, dll]"
  },
  "style": "[candid, posed, lifestyle, dll]"
}
```

### Untuk Video Motion (Seedance 2.0 / Sora 2)

```
[Subject] [action — holds, walks, turns, smiles],
[detail movement — hand gestures, head tilt],
[environment interaction — sits on couch, leans on counter],
[camera movement — static, slow pan, slight zoom],
[lighting — golden hour, soft indoor, bright studio],
[mood — warm, energetic, calm, dramatic]
```

**Tip dari @IamEmily2050:** Gunakan **Kimi K2** untuk draft prompt, lalu refine manual. Emily menggunakan workflow: draft di Grok 4 → refine di Kimi K2 → generate di target platform.
