## Glossary

| Istilah | Arti |
|---------|------|
| **UGC** | User-Generated Content — konten yang terlihat dibuat user biasa |
| **Lip sync** | Sinkronisasi gerakan bibir dengan audio |
| **Prompt** | Instruksi text yang diberikan ke AI |
| **Context Engineering** | Pemberian context lengkap ke AI untuk hasil presisi |
| **Image-to-Video** | Konversi gambar statis jadi video |
| **Composite** | Menggabungkan elemen visual dari sumber berbeda |
| **Hook** | 3 detik pertama video yang menarik perhatian |
| **CTA** | Call to Action — ajakan untuk melakukan sesuatu |
| **ROAS** | Return on Ad Spend — metrik efektivitas iklan |
| **Seed** | Angka random yang menentukan output AI |
