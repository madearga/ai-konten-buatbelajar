# Workflow Branded Content — Step by Step

## Overview

Workflow ini untuk produksi **branded content** — konten premium yang on-brand, untuk brand awareness, product launch, dan social media campaigns.

---

## End-to-End Branded Campaign

<Steps>

### Step 1: Creative Brief & Concept

Sebelum touch AI tools, siapkan **creative brief**:

| Element | Isi |
|---------|-----|
| **Objective** | Brand awareness? Product launch? Engagement? |
| **Target audience** | Demografi, interest, pain points |
| **Key message** | 1 kalimat inti pesan |
| **Visual style** | Minimal, luxury, playful, energetic? |
| **Platform** | TikTok? Instagram? YouTube? Multi-platform? |
| **Format** | Video? Carousel? Story? Reel? |
| **Budget** | Berapa banyak konten yang perlu diproduksi? |

### Step 2: Visual Direction (Midjourney / Flux)

Generate **moodboard** untuk visual direction:

**Prompt Midjourney untuk moodboard:**
```
minimal moodboard layout, luxury skincare brand, 
soft pastel palette, natural textures, clean typography,
4 product photography styles, editorial aesthetic
--ar 16:9 --v 7 --style raw
```

Generate 4-8 variasi → pilih direction yang paling on-brand.

### Step 3: Brand Assets

<Tabs items={['Product Shots', 'Brand Graphics', 'Social Templates']}>
  <Tab>
    ### Product Shots dengan AI
    
    1. **Upload product image** ke Midjourney sebagai image prompt
    2. **Generate variations** dengan berbagai background dan lighting
    3. **Composite** jika perlu detail spesifik (Photoshop / Canva)
    
    **Prompt template:**
    ```
    [product] on [surface], [lighting], 
    [color palette] background, 
    professional product photography, 
    [mood] aesthetic --ar 4:5 --style raw
    ```
  </Tab>
  <Tab>
    ### Brand Graphics
    
    Gunakan AI untuk:
    - **Logo variations** → Midjourney / Dreamina
    - **Color palette exploration** → Generate dengan mood keywords
    - **Pattern & texture** → Seamless patterns untuk packaging/web
    - **Typography exploration** → AI-generated text layouts
  </Tab>
  <Tab>
    ### Social Media Templates
    
    1. Generate base image dengan AI
    2. Tambahkan text overlay di Canva / Figma
    3. Buat template reusable untuk konsistensi
    
    **Format per platform:**
    - **Instagram Feed:** 1080x1080 (1:1) atau 1080x1350 (4:5)
    - **Instagram Story/Reels:** 1080x1920 (9:16)
    - **TikTok:** 1080x1920 (9:16)
    - **YouTube Thumbnail:** 1280x720 (16:9)
  </Tab>
</Tabs>

### Step 4: Video Production

<Tabs items={['Concept Video', 'Product Demo', 'Brand Story']}>
  <Tab>
    ### Concept / Hero Video
    
    **Workflow:** Storyboard (Midjourney) → Video (Sora 2 / Seedance 2.0) → Edit
    
    1. Generate 6-12 storyboard frames di Midjourney
    2. Konversi setiap frame ke video clip (5-10 detik per clip)
    3. Edit dan gabungkan di DaVinci Resolve / CapCut
    4. Tambahkan music, voice over, dan text overlay
    
    **Durasi target:** 30-60 detik untuk hero video
  </Tab>
  <Tab>
    ### Product Demo Video
    
    **Workflow:** Product shots → Video animation → Edit
    
    1. Generate product shots dari berbagai angle
    2. Gunakan **image-to-video** untuk animasi (rotate, zoom, reveal)
    3. Tambahkan text overlay untuk feature highlights
    4. Background music + voice over
    
    **Durasi target:** 15-30 detik
  </Tab>
  <Tab>
    ### Brand Story Video
    
    **Workflow:** Script → Storyboard → Character → Video → Edit
    
    1. Tulis narasi brand story (gunakan AI copywriter jika perlu)
    2. Generate storyboard visual (Midjourney)
    3. Jika butuh karakter → Nano Banana 2 untuk consistent character
    4. Video generation per scene (Sora 2 / Seedance 2.0)
    5. Edit + music + voice over + color grading
    
    **Durasi target:** 60-90 detik
  </Tab>
</Tabs>

### Step 5: Multi-Platform Adaptation

Ubah satu konten menjadi **multi-platform assets**:

| Source | Adaptasi | Tools |
|--------|----------|-------|
| Hero video (16:9) | Reel (9:16) | CapCut auto-reframe |
| Hero video | 15-sec highlight | Manual cut |
| Product shots | Carousel post | Canva batch resize |
| Storyboard frames | Static social posts | Direct export |
| Voice over script | Caption/subtitle | CapCut auto-caption |

### Step 6: Schedule & Deploy

- **Scheduling tool:** Meta Business Suite, Later, atau Buffer
- **Best practices:** Test waktu posting, A/B test thumbnail, monitor performance
- **Iteration:** Gunakan data performa untuk improve konten berikutnya

</Steps>
