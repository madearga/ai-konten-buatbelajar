# Setup Tools Web

Sebelum mulai, kamu perlu **daftar akun** di tools yang akan dipakai. Semua berbasis web — cukup browser.

---

## Detail Setup per Tool

<Tabs items={['Image Tools', 'Video Tools', 'UGC Tools', 'Audio Tools']}>
  <Tab>
    ### Image Generation

    **Dreamina** (Gratis):
    1. Buka dreamina.com
    2. Sign up dengan Google/email
    3. Klik "Create" → tulis prompt → generate
    - Cocok untuk: poster, social media, eksplorasi awal
    - Limit: 10-20 generate/hari (free tier)

    **Freepik** (Freemium):
    1. Buka freepik.com
    2. Sign up
    3. Gunakan "AI Image Generator" atau search stock
    - Cocok untuk: stock photos + AI generation
    - Free tier: 10 AI generate/hari

    **Pixazo** (Pay-per-use):
    1. Buka pixazo.ai
    2. Daftar dan dapatkan API key
    3. Akses 30+ model termasuk Nano Banana Pro, Flux, Seedance
    - Cocok untuk: karakter AI premium, multi-model access
    - Biaya: ~$0.05-0.50 per gambar

    **fal.ai** (Pay-per-use):
    1. Buka fal.ai
    2. Daftar dan top-up credits
    3. Akses banyak model AI via web atau API
    - Cocok untuk: alternative ComfyUI tanpa install
    - Biaya: mulai dari $0.01/gambar
  </Tab>
  <Tab>
    ### Video Generation

    **Kling AI** (Free tier):
    1. Buka klingai.com
    2. Daftar dengan email/Google
    3. Free: 66 credits/hari (~6 video pendek)
    - Cocok untuk: quick social video, testing
    - Max: 5-15 detik per video

    **Sora 2** via ChatGPT Pro:
    1. Buka chatgpt.com
    2. Upgrade ke ChatGPT Pro ($200/bulan)
    3. Akses Sora dari menu
    - Cocok untuk: video premium, sinematik
    - Max: 60 detik per video

    **Seedance 2.0** (via Dreamina atau Pixazo):
    - Akses via dreamina.com atau pixazo.ai
    - Cocok untuk: realistic video + lip sync
    - Biaya: $10-30 per video
  </Tab>
  <Tab>
    ### UGC Ads

    **CreateUGC** (Free trial 7 hari):
    1. Buka createugc.ai
    2. Daftar → langsung dapat 7 hari free
    3. Paste URL produk → generate UGC ads
    - Cocok untuk: bulk UGC production
    - Setelah trial: $29/bulan (Starter)

    **Topview** (Free tier):
    1. Buka topview.ai
    2. Daftar
    3. Upload produk / paste URL → generate
    - Cocok untuk: quick UGC, market Indonesia
    - Support Bahasa Indonesia

    **Higgsfield** (Berbayar):
    1. Buka higgsfield.ai
    2. Pilih plan ($20-100/bulan)
    3. Akses Marketing Studio
    - Cocok untuk: full brand campaign, virtual try-on
  </Tab>
  <Tab>
    ### Audio

    **ElevenLabs** (Free tier):
    1. Buka elevenlabs.io
    2. Daftar → free: 10,000 characters/bulan
    3. Pilih voice → tulis/paste script → generate
    - Cocok untuk: voice over video, karakter berbicara
    - 30+ bahasa termasuk Bahasa Indonesia

    **Suno** (Free tier):
    1. Buka suno.com
    2. Daftar
    3. Tulis deskripsi musik → generate
    - Cocok untuk: background music original
    - Free: ~10 lagu/bulan
  </Tab>
</Tabs>

## Latihan

1. Daftar akun di **semua** free tools di checklist atas
2. Di Dreamina, generate 1 gambar dengan prompt: "Indonesian woman holding skincare serum bottle, soft natural lighting, beauty photography"
3. Di Kling, generate 1 video 5 detik dari gambar yang kamu buat
4. Simpan hasilnya — akan dipakai di modul berikutnya
