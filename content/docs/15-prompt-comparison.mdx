## Keputusan: Pilih Format yang Tepat

<Callout type="info" title="Rekomendasi Alur">
Untuk FMCG/Beauty/Health brands:
1. **Campaign/Branded Content** → Gunakan **Detailed (Panjang)** atau **JSON Terstruktur**
   - Perlu storytelling, cinematic lighting, polished finish
   - Hasil: Brand yang kredibel dan premium

2. **UGC Ads** → Gunakan **Detailed (Panjang)** untuk testimonial / demo
   - Perlu natural setting, genuine vibe, produk terlihat jelas
   - Format "Mix" untuk variasi batch production

3. **Social Media** → **Basic** atau **Mix** cukup
   - Kecepatan lebih penting daripada kualitas super detail
   - Bisa gunakan "auto-generate" features di Nano Banana Pro

4. **Batch Production** → Gunakan **JSON Terstruktur** atau **Mix**
   - Memudahkan A/B testing dengan struktur yang konsisten
   - Generate 10-20 variasi sekaligus, filter yang terbaik

**Kapan pindah ke JSON Terstruktur:** 
Jika kamu butuh **workflow automation** (auto-generate 10+ variasi, combine otomatis), gunakan format JSON untuk setiap shot dalam sequence terpisah.
</Callout>
