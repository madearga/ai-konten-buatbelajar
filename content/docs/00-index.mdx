# Panduan AI Content Creation — Internal Training

**Dibuat untuk:** Content Team (1-5 orang, pemula total)
**Fokus industri:** FMCG, Beauty, Health
**Format:** Workshop 1-2 hari + referensi mandiri

---

## Apa yang Akan Kamu Pelajari?

Di akhir panduan ini, kamu bisa:

- 🎯 **Memilih tool AI yang tepat** untuk setiap kebutuhan konten
- 🧑 **Membuat karakter AI** yang konsisten (AI influencer/model)
- ✍️ **Menulis prompt** yang menghasilkan output sesuai keinginan
- 🎬 **Memproduksi video AI** untuk UGC ads dan branded content
- 📱 **Mengintegrasikan produk** ke konten AI (skincare, lipstick, supplement)
- ✂️ **Final editing** sampai siap deploy ke TikTok/Instagram/Meta Ads

## Tools yang Akan Kita Pakai

Semua tools di panduan ini **berbasis web** — tidak perlu install software berat atau punya GPU. Cukup browser dan koneksi internet.

| Kategori | Tools | Biaya |
|----------|-------|-------|
| **Karakter AI** | Nano Banana Pro (via Pixazo/fal.ai) | $0.05-0.50/gambar |
| **Video AI** | Sora 2, Seedance 2.0, Kling | $0-200/bulan |
| **UGC Cepat** | CreateUGC, Topview | $19-99/bulan |
| **Audio** | ElevenLabs, Suno | Free tier tersedia |
| **Image** | Dreamina, Freepik, Mitte | Free tier tersedia |
| **All-in-One** | Pixazo, fal.ai | Pay-per-use |
