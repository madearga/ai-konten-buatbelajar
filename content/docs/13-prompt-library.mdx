## Prompt Fundamental (dari Yaroflasher)

### Pelajaran dari Video "Omni Pro Workflow"

Yaroflasher melakukan **4 jenis test prompt** untuk Seedance 2:

<Tabs items={['Basic (Pendek)', 'Detailed (Panjang)', 'JSON Structure', 'Mix (Text + JSON)']}>
  <Tab>
    ### Basic Prompt — Kurang Kontrol

    ```
    glasses on a gray background, rotating
    ```

    **Hasil:** "Looked very boring" — model tidak punya cukup informasi. Hanya cocok untuk eksplorasi awal.

    **Kapan pakai:** Testing awal, brainstorming cepat
  </Tab>
  <Tab>
    ### Detailed Prompt — Lebih Baik

    Dibuat dengan bantuan ChatGPT — menambahkan deskripsi scene, camera angles, timing, dan animation logic.

    ```
    A sleek pair of futuristic glasses rotates slowly on a neutral gray surface.
    Camera starts with a wide shot, then gradually zooms into a close-up of the frame.
    Soft studio lighting from the left, with subtle reflections on the metal surface.
    The glasses complete one full rotation over 8 seconds.
    Shot on Arri Alexa Mini, 35mm lens, cinematic depth of field.
    ```

    **Hasil:** Signifikan lebih baik daripada basic. Tapi masih kurang kontrol untuk motion spesifik.
  </Tab>
  <Tab>
    ### JSON Prompt — Struktur Terorganisir

    ```json
    {
      "shot": "product_showcase",
      "subject": "futuristic glasses, metal frame, dark lenses",
      "camera": {
        "start": "wide shot, 45 degree angle",
        "movement": "slow orbit around the object",
        "end": "close-up on temple hinge",
        "lens": "100mm macro, shallow depth of field"
      },
      "lighting": "dramatic studio lighting, key light from upper left, fill light from right",
      "environment": "neutral dark gray background, subtle gradient",
      "duration": "8 seconds",
      "style": "luxury product photography, premium commercial aesthetic"
    }
    ```

    **Hasil:** Setara dengan detailed text prompt. Keuntungan: lebih mudah di-modify per bagian.
  </Tab>
  <Tab>
    ### Mix (Text + JSON) — Kontrol Maksimal

    ```
    Luxury product showcase of futuristic glasses.
    Premium commercial aesthetic with cinematic lighting.
    The glasses should catch light reflections beautifully.
    ```
    ```json
    {
      "camera": {
        "movement": "slow dolly in, then orbit clockwise",
        "speed": "slow and smooth"
      },
      "timing": {
        "0-3s": "wide shot, glasses centered",
        "3-6s": "camera orbits, showing side profile",
        "6-8s": "close-up on lens surface reflections"
      }
    }
    ```

    **Hasil:** "Most controlled setup" menurut Yaroflasher. Text handle atmosphere + style, JSON handle structure + timing.
  </Tab>
</Tabs>

<Callout type="warning" title="Key Insight dari Yaroflasher">
"Perbedaan antara format prompt tidak terlalu signifikan. Yang lebih penting adalah **randomness** (seed), **input images**, dan **kualitas penulisan prompt secara umum**. Short prompt perform jauh lebih buruk — tidak cukup kontrol. Detailed description selalu memberikan hasil lebih baik."
</Callout>

## First Frame / Last Frame Technique

*Credit: Framer (via Theoretically Media)*

**Konsep:** Berikan 2 gambar ke Seedance — gambar pertama (awal) dan gambar terakhir (akhir). AI generate animasi di antara keduanya.

```
Show me what happens in between. Use multiple camera angles.
```

**Cara pakai untuk product storytelling:**
1. **Frame 1:** Produk masih dalam kemasan / tutup tertutup
2. **Frame 2:** Produk sudah terbuka / sedang dipakai
3. Seedance generate "unboxing" / "reveal" animation secara otomatis

## Workflow: Prompt → Video (Yaroflasher Pipeline)

<Steps>

### 1. Buat Product Assets
Generate multiple angles produk dengan Nano Banana Pro. Gunakan prompt macro photography di atas.

### 2. Test Prompt (Iterate!)
Jangan langsung commit. Test 3-5 variasi prompt:
- Basic text → lihat baseline
- Detailed text → tambah kontrol
- JSON → struktur terorganisir
- Mix → kontrol maksimal

### 3. Tambah Video Reference (Opsional)
Kalau perlu kontrol motion yang presisi, buat 3D animation sederhana (Cinema 4D / Blender) sebagai video reference. Tambahkan di prompt:
```
Use video reference for camera and object positioning only.
Take the shape and materials of [PRODUCT] from the image, not the video.
```

### 4. Generate & Select
Generate 5-10 variasi per shot. Pilih yang terbaik, buang sisanya. **Randomness (seed) matters!**

### 5. Edit Final
Gabungkan best clips di CapCut. Tambah audio + subtitle.

</Steps>

## Tips Prompt dari Video Terbaru Yaroflasher

| Tip | Penjelasan |
|-----|-----------|
| **Iterate in 1K, deliver in 4K** | Test di resolusi rendah dulu untuk hemat credits. Upscale hanya "winner" |
| **Speed is a tool** | Speed up video 500-800% bikin rhythm dan hide glitch |
| **First-frame rule** | Orientasi gambar pertama menentukan aspect ratio seluruh generation |
| **Short prompt = bad** | Prompt pendek selalu perform lebih buruk |
| **Randomness > format** | Perbedaan format (text vs JSON) tidak signifikan. Seed dan input images lebih penting |
| **Video reference conflict** | Model kadang ignore video reference kalau ada terlalu banyak elemen tambahan |
| **No reference = more creative** | Hapus video reference untuk efek organik, tapi hilang kontrol presisi |
