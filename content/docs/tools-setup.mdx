## Hardware Requirements

### Local Rendering (ComfyUI / Flux)

| Task | Minimum GPU | Recommended | VRAM |
|------|-------------|-------------|------|
| **Image generation** | RTX 3060 (12GB) | RTX 3090 (24GB) | 12-24GB |
| **LoRA training** | RTX 3060 (12GB) | RTX 3090 (24GB) | 12-24GB |
| **Video generation** | RTX 3090 (24GB) | RTX 4090 (24GB) | 24GB |
| **Multiple workflows** | RTX 3090 (24GB) | RTX 4090 (24GB) | 24GB |

### Cloud GPU Alternatives

| Provider | GPU | Harga | Best For |
|----------|-----|-------|----------|
| **RunPod** | RTX 3090 | ~$0.30/hr | Image workflows |
| **RunPod** | RTX 4090 | ~$0.60/hr | Video workflows |
| **Vast.ai** | RTX 3090 | ~$0.20/hr | Budget option |
| **Vast.ai** | A100 | ~$1.00/hr | Heavy video |
| **Google Colab** | T4 (free) | Free | Testing |
| **Google Colab Pro** | V100/A100 | $10-50/hr | Occasional use |

<Callout type="info" title="Local vs Cloud Decision">
  - **Pilih Local** jika: kamu generate **>100 images/hari** atau **>10 videos/hari** — investasi GPU lebih hemat
  - **Pilih Cloud** jika: kamu baru mulai, **produksi tidak rutin**, atau butuh GPU besar hanya sesekali
  - **Hybrid (rekomendasi):** Local untuk image generation rutin, Cloud untuk video generation yang berat
</Callout>
