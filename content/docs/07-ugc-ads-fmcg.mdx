## 2 Workflow UGC

<Tabs items={['Quick: CreateUGC', 'Pro: Manual']}>
  <Tab>
    ### Quick Workflow — CreateUGC (30 menit)

    Untuk produksi cepat, pakai **CreateUGC**. Dari URL produk langsung jadi UGC ads.

    <Steps>

    ### Input Produk
    Paste URL produk dari Shopify/WooCommerce/marketplace. AI auto-extract info produk, gambar, dan selling points.

    ### Pilih Avatar
    Filter berdasarkan:
    - Gender: sesuai target audience
    - Usia: 20-30 untuk skincare/beauty, 25-40 untuk supplement
    - Etnis: Southeast Asian untuk market Indonesia
    - Style: casual untuk UGC, professional untuk branded

    ### Tulis Script (3 Hook Variations)
    ```
    Hook 1: "Eh, kamu punya masalah kulit kering juga?"
    Hook 2: "Dulu aku nggak percaya sama skincare..."
    Hook 3: "Cek ini — 2 minggu pakai, hasilnya gila!"
    ```

    ### Generate
    Klik generate. Tunggu 5-10 menit. Download hasilnya.

    ### Quality Check
    - ✅ Lip sync akurat
    - ✅ Produk terlihat natural
    - ✅ Hook kuat di 3 detik pertama
    - ✅ CTA jelas

    ### Deploy
    Upload ke TikTok Ads / Meta Ads. Test 3 hook variations dengan budget kecil ($5-10/hari per variasi).

    </Steps>
  </Tab>
  <Tab>
    ### Pro Workflow — Manual (2-3 jam)

    Untuk kualitas premium dan kontrol penuh. Menggabungkan semua tools.

    <Steps>

    ### Step 1: Script + Shot List
    Tulis script dan breakdown jadi shots:

    ```
    SHOT 1 (3s): Hook — close-up wajah, ekspresi surprised
    SHOT 2 (5s): Problem — medium shot, gestur frustrated
    SHOT 3 (8s): Solution — memegang produk, aplikasi
    SHOT 4 (3s): Result — smile, before/after
    SHOT 5 (3s): CTA — tampilkan produk + link
    ```

    ### Step 2: Generate Character
    Pakai Nano Banana Pro (Modul 04). Generate karakter untuk setiap shot.
    Pastikan **konsisten**: same subject description, hanya beda pose.

    ### Step 3: Integrate Product
    Pakai product integration technique (Modul 09 detail).
    Quick method: generate karakter dulu, lalu composite produk di Canva/CapCut.

    ### Step 4: Image-to-Video
    Setiap shot → generate video 3-8 detik via Seedance 2.0 atau Kling.

    ### Step 5: Lip Sync
    Generate voice over via ElevenLabs.
    Upload video + audio ke Seedance untuk lip sync.

![Contoh UGC ads green screen — wanita Asia memegang vitamin C serum di pantai tropis, vertical 9:16 format, social media influencer style](/images/examples/arena-skincare-green-screen-ugc.png)

<Callout type="tip" title="Green Screen UGC">
Gambar di atas dihasilkan dari prompt "Product photography of a skincare brand green screen UGC ad, young Asian woman with fresh glowing skin holding a vitamin C serum bottle, vibrant tropical beach background composited behind her" — menggunakan **gemini-3.1-flash-image-preview** di arena.ai.

Workflow: generate karakter di green screen background → edit background di CapCut → composite dengan lokasi/brand setting yang dikehendaki.
</Callout>

    ### Step 6: Edit Final
    Gabungkan semua clips di **CapCut** (gratis):
    - Arrange clips sesuai shot list
    - Tambah subtitle (80% viewer tanpa suara)
    - Tambah background music (Suno atau royalty-free)
    - Export format 9:16 (1080x1920)

    </Steps>
  </Tab>
</Tabs>

## Script Template UGC Ads

### Template: Problem-Solution (Skincare)

```
[HOOK 3s]: "3 detik! Aku mau tunjukin sesuatu..."
[PROBLEM 5s]: "Dulu kulit aku kering, kusam, nggak pede keluar rumah"
[SOLUTION 8s]: "Tapi coba liat sekarang — 2 minggu pakai [PRODUK],
              kulit aku berubah total. Caranya gampang..."
[RESULT 3s]: "Ini before-after aku. Serious."
[CTA 3s]: "Klik link di bio buat coba sendiri! 🔥"
```

### Template: Testimonial (Supplement)

```
[HOOK 3s]: "Aku sempet ragu, tapi..."
[STORY 5s]: "Setelah 1 bulan pakai [PRODUK], energi aku naik drastis"
[DEMO 8s]: "Tiap pagi cuma perlu 1 capsule — gitu aja. Rasanya juga enak"
[RESULT 3s]: "Beneran kerasa bedanya, bukan cuma omong kosong"
[CTA 3s]: "Cek di link bio, ada promo juga! 💪"
```

### Template: Taste Test (Snack/F&B)

```
[HOOK 3s]: "Snack ini bikin aku autis..."
[REACTION 5s]: "Serious, rasanya gila banget — renyah, gurih, nagih!"
[DEMO 8s]: "Buka kemasannya, liat texturenya — Crunch! Langsung Add to Cart"
[ENDORSE 3s]: "Rating 4.9 di marketplace, bukan cuma aku yang ketagihan"
[CTA 3s]: "Grab sekarang sebelum sold out! 🔥"
```

## Checklist Sebelum Deploy

<Callout type="warning" title="QA Wajib">
- [ ] Hook kuat di 3 detik pertama
- [ ] Produk terlihat jelas dan natural
- [ ] Script sesuai brand tone
- [ ] Lip sync akurat
- [ ] Subtitle/captions ditambahkan
- [ ] CTA jelas dan actionable
- [ ] Format 9:16 (TikTok/Reels)
- [ ] Durasi 15-30 detik
- [ ] Minimal 3 variasi hook untuk testing
</Callout>
